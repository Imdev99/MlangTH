# MlangTH - Minecraft Resource Pack

มอดแปลภาษาไทยและเปลี่ยนฟอนต์สำหรับ Minecraft Bedrock Edition 
พัฒนาโดย **Judo_2310**

## 📝 เครดิต (Credits)

* **ผู้พัฒนาโปรเจกต์:** [Judo_2310](https://github.com/imdev99)
* **การแปลภาษา (Translation):** จัดทำโดย Judo_2310 (เน้นคำแปลที่ใช้งานง่าย และรองรับบล็อกพิเศษ/บล็อกคำสั่ง)
* **ฟอนต์ (Font):** [ใส่ชื่อฟอนต์ที่คุณใช้] (เช่น "ฟอนต์จากโครงการ... หรือชื่อผู้สร้างฟอนต์")
* **ระบบช่วยเรียบเรียงโค้ด:** พัฒนาและตรวจสอบโครงสร้างไฟล์ .lang ร่วมกับ Gemini AI
