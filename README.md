# MlangTH (MLTH) - Minecraft Resource Pack 🇹🇭

Resource Pack ภาษาไทยสำหรับ Minecraft Bedrock Edition (PE) ที่เน้นความสะอาดตาและการแปลที่ถูกต้องสำหรับสาย Technical และ Builder พัฒนาโดย **Judo_2310**

## 🌟 จุดเด่นของโปรเจกต์ (Features)
- **แปลชื่อบล็อกพิเศษ:** เปลี่ยนชื่อบล็อกที่ระบบเดิมแปลไม่ครบ เช่น 
    - `Barrier` -> **บล็อกล่องหน**
    - `Command Blocks` -> **บล็อกคำสั่ง** (ตัดวงเล็บสีออกเพื่อความสะอาดตา)
- **ระบบบล็อกแสง:** เพิ่มชื่อภาษาไทยให้กับ `Light Block` ครบทุกระดับ (0-15)
- **คงเอกราชสายเทคนิค:** ไอเทมเฉพาะทางอย่าง `Beacon` และ `Conduit` จะคงชื่อภาษาอังกฤษไว้ตามความนิยมของชุมชนผู้เล่น
- **ฟอนต์อ่านง่าย:** ใช้ฟอนต์ Sarabun ที่ให้ความรู้สึกทันสมัยและเป็นทางการ
